🔧 Creating Your Own Mesh Links

To create a link to another MeshWeb node, use this format:

<a href="mesh://[NODE_ID]/[PAGE_PATH]">Link Text</a>

Where:

  • [NODE_ID] = The 8-character hex ID of the target node (e.g., "ff1a2b3c")
  • [PAGE_PATH] = The path to the page (e.g., "index.html" or "pages/about.html")

Example:

<a href="mesh://ff1a2b3c/welcome.html">Visit Node ff1a2b3c</a>

📍 Finding Node IDs

To find the node ID of a MeshWeb host:

  1. Connect your companion to the mesh network
  2. Check the serial monitor of the host node (shows at boot)
  3. Or look at the node list on the companion web interface
💡 Pro Tip: You can create entire networks of linked pages across multiple MeshWeb nodes, building a true decentralized web over LoRa!
